# Type Comportments

**Type Comportments** é uma série de experimentos interativos em p5.js sobre tipografia, física e comportamento visual.

A proposta é investigar o que acontece quando uma letra deixa de ser apenas signo e passa a operar como sistema: partícula, campo, mola, ruído, rastro, névoa, constelação ou superfície óptica.

## Conceito

Este projeto parte de uma pergunta simples:

> E se a tipografia pudesse se comportar?

Cada experimento traduz um conceito físico, matemático ou computacional em uma experiência visual direta. Em vez de explicar teoria de forma abstrata, o projeto usa letras, palavras e blocos de texto como matéria sensível.

## Experimentos

### Nível 01 — Palavras

- **01 Latência** — distância, repulsão e campo
- **02 Tensão** — lei de Hooke, mola e oscilação
- **03 Ruído** — Perlin noise e turbulência
- **04 Traço** — contorno, curva e escrita
- **05 Eco** — rasterização, grid e deslocamento

### Nível 02 — Blocos de texto

- **06 Névoa** — ruído como percepção
- **07 Constelação** — grafo, proximidade e relação
- **08 Erosão** — vento, impermanência e dispersão

### Nível 03 — Forma + texto

- **09 Lente** — projeção esférica e distorção óptica
- **10 Halftone** — luz, matriz e input visual
